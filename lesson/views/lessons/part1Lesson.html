Space Avenger

Good job, Code Champion! You just learned about for loops! In Lesson 2, we are going to discover while loops.

In this lesson, we are going to space! You're going to program a Space Avenger to blast the alien invaders.

In order for the Space Avenger to beat the aliens, you are going to make some basic functions that will work together to defeat them.

Code normally execudes in the order it's written. Loops allow you to repeat a block of code multiple times without having to re-type it.

First, we start a loop with the while keyword. This tells your program WHILE something is true, repeat the body of the loop.

For now we want our loops to continue forever, so we'll use a while-true loop. Because true is always true!

// Start the while-true loop with
while(true) {
// Anything between the { and } is considered INSIDE the loop.
  this.moveRight();
  this.moveLeft();
}

In this game, each level will need different functions in the while-true loop to defeat the alien invaders.
